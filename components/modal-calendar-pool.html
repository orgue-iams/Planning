Calendriers des utilisateurs (pool)

Créez d’abord chaque calendrier dans Google Agenda (compte orgue.iams), puis ajoutez-le un par un avec son ID (vous pouvez coller l’URL d’intégration : l’ID est extrait du paramètre src). Partagez chaque calendrier secondaire avec l’e-mail du compte de service / OAuth utilisé par calendar-bridge, avec « Modifier les événements » — comme pour l’agenda principal — sinon la synchro miroir renvoie « Not Found ». Les comptes élève / prof / admin reçoivent le prochain créneau libre du pool. Si le pool est vide, le profil affiche POOL_SATURATED.
